## 🌟 About

**WhatsApp Classic 2006** is a Y2K-inspired messaging web application that imagines how WhatsApp would have looked if it had been released in **2006**.

Inspired by **Windows XP**, **MSN Messenger**, **Yahoo! Messenger**, **Internet Explorer 6**, and the **Frutiger Aero** design movement, the project recreates the nostalgic experience of early internet messaging while using modern technologies for authentication, real-time communication, and responsive performance.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/whatsapp-classic-2006.git

cd whatsapp-classic-2006
```

---

## Install Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## Install Backend

```bash
cd backend

npm install

npm run dev
```

---

# ⚙ Environment Variables

Create a `.env` file inside the backend directory.

```env
PORT=5000

MONGO_URI=your_mongodb_uri

JWT_SECRET=your_secret_key

CLIENT_URL=http://localhost:3000
```
